# EnvX - AI智能环境配置助手

EnvX是一个利用AI技术智能完成开发环境配置的Agent，能够根据用户需求或项目README文档等，一键适配并搭建所需的开发环境。

## 🌟 主要特性

- 🤖 AI驱动的环境配置
- 🚀 一键自动化安装
- 📝 智能解析项目需求
- 🔍 自动错误诊断与修复
- 👨‍🏫 交互式配置指导

## 🛠 安装

```bash
pip install envx
```

## 🚀 快速开始

```python
from envx import EnvXAgent

# 初始化EnvX Agent
agent = EnvXAgent()

# 根据项目需求配置环境
agent.configure("path/to/your/project")
```
